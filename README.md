# 🌳 Roots & Rise — WeCoded 2026 Frontend Art

> *"every branch was once a root"*

A Frontend Art submission for the [2026 WeCoded Challenge](https://dev.to/challenges/wecoded-2026) on DEV.to.

**Roots & Rise** is an animated SVG artwork built entirely with HTML, CSS, and vanilla JavaScript. It depicts a living tree whose anatomy maps the story of gender equity in tech — what grounds us, who carried us, and what we are still growing toward.

## 🎨 What's in the Tree

| Element | Meaning |
|---|---|
| 🌱 **Roots** (underground) | Persistence, community, legacy, courage, solidarity, resilience — the invisible foundations |
| 🌿 **Branches** | Pioneer names etched into the bark — Ada Lovelace, Grace Hopper, Katherine Johnson, Margaret Hamilton, and more |
| 🌳 **Canopy** | Shared shelter — the expansive space equity makes possible for everyone |
| 🌸 **Blossoms & particles** | Voices, ideas, and futures in motion — rising forever |
| ☀️ **Gold crown** | Not a ceiling — a sun. The highest branches reach toward light. |

---

## 🛠 Tech Stack

- Pure **HTML + CSS + SVG** — zero frameworks, zero dependencies
- **SVG `stroke-dasharray` animation** — branches draw themselves on load
- **`requestAnimationFrame` sway physics** — each particle has independent amplitude, period & phase
- **CSS custom properties** — per-particle `--dur` and `--delay` for staggered rise animation
- **SVG gradients** — `<linearGradient>` and `<radialGradient>` for atmosphere and depth
- **Google Fonts** — Playfair Display + DM Mono for botanical editorial typography
